# ДОБАВЛЕНИЕ ЭЛЕМЕНТОВ В НАЧАЛО СВЯЗНОГО СПИСКА

Сложность: 2 из 10

Добавьте в класс `List` метод `prepend`, который будет принимать значение и добавлять его в начало связного списка.

**Пример использования**:

```python
lst = List()
lst.append("A")
lst.append("B")
lst.prepend("C")
lst.prepend("D")
print(lst)
[D, C, A, B]
```
